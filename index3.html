Подключение по FTP и SSH ( Visual Studio Code )

Для подключения по FTP необходимо:

1. Установить Remote FS .

2. Open User Settings:

On Windows/Linux - File > Preferences > Settings
On macOS - Code > Preferences > Settings

3. Add your remote configs to "remotefs.remote" in your User Settings. 

                {
                    "remotefs.remote": {
                
                        "site": {
                            "scheme": "ftp",
                            "host": "",
                            "username": "",
                            "password": ""
                          }
                        },
                        "php.validate.executablePath": "c:/php7/php.exe",
                        "remote.autoForwardPorts": false,
                        "remote.extensionKind": {
                
                          "pub.name": [
                            "ui"
                          ]
                        }
                }

4. Cmd+Shift+P open command palette(Ctrl+Shift+P on Windows/Linux)
5. Run Remote FS: Add Folder to Workspace command.

Для подключения по SSH необходимо:

1. Открыть терминал.
2. Выполнить команду: 

                ssh login@serverAddress
                login - логин FTP-аккаунта
                serverAddress - Адрес FTP-сервера

3. Ввести пароль от FTP-аккаунта
